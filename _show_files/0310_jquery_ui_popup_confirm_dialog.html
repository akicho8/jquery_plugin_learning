31. ポップアップで選択させたい

本当に18才以上ですか？

HTML

<div id="main" title="確認">
  <p>本当に18才以上ですか？</p>
</div>

JavaScript

$("#main:ui-dialog").dialog("destroy");
$("#main").dialog({
  modal: true,
  buttons: {
    "はい": function() {
      $(this).dialog("close");
    },
    "いいえ": function() {
      $(this).dialog("close");
    },
    "秘密": function() {
      $(this).dialog("close");
    },
  }
});

MEMO

jquery-uiのダイアログはいくつでも選択ボタンをつけれるので便利
