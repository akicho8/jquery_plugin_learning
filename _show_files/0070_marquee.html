7. マーキー使用時にマウスオーバーで停止したりするには？

親譲りの無鉄砲で小供の時から損ばかりしている。

HTML

<marquee behavior="scroll" scrollamount="1" direction="left" width="160" id="main">
  親譲りの無鉄砲で小供の時から損ばかりしている。
</marquee>

JavaScript

$("#main").marquee("pointer").mouseover(function(){
  $(this).trigger("stop");
}).mouseout(function () {
  $(this).trigger("start");
});

MEMO
