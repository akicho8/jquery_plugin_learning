72. TinyMCEの使い方

HTML

<textarea id="body">
</textarea>

JavaScript

$("#body").tinymce({
  script_url: 'assets/tinymce/jscripts/tiny_mce/tiny_mce.js',
});

MEMO

script_url オプションを指定しないと tinymce not defined になるので注意
サンプルを実行するには http:// からアクセスできるようにして http://jquery_plugin_learning/assets/tinymce/examples/index.html を参照する
