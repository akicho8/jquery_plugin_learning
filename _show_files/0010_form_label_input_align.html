1. labelとinputを横に並べたときでもinputの列を揃えるには？

HTML

<form action="#" id="main">
  <p class="field"><label>題名</label><input type="text" /></p>
  <p class="field"><label>メールアドレス</label><input type="text" /></p>
  <p class="field"><label>サイト</label><input type="text" /></p>
</form>

JavaScript

var max = 0;
$("label").each(function(){
  if ($(this).width() > max) {
    max = $(this).width();
  }
});
$("label").width(max);
