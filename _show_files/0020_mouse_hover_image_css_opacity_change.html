2. マウスオーバーしたときに画像の輝度を変更するには？

あいうえお

HTML

<div id="main">
  <img src="assets/rails.png" />
  <span>あいうえお</span>
  <button>あいうえお</button>
</div>

JavaScript

$("#main *").hover(
  function(){
    $(this).css({opacity: 0.6});
  },
  function(){
    $(this).css({opacity: 1.0});
  });
